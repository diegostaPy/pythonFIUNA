# Fundamentos de Programación para Ingeniería  

## 📌 Descripción del Curso  
Este curso introduce a los estudiantes de ingeniería en los principios fundamentales de la programación utilizando **Python**. A lo largo del curso, se desarrollarán habilidades en resolución de problemas computacionales, estructuras de datos básicas y manipulación de datos.  

## 📚 Contenido del Curso  
- 🔹 Introducción a Python y su sintaxis  
- 🔹 Control de flujo: condicionales y bucles  
- 🔹 Funciones y modularización del código  
- 🔹 Manejo de estructuras de datos (listas, tuplas, diccionarios)  

## 📂 Material Disponible  
Los siguientes recursos serán publicados regularmente:  
✅ **Ejercicios resueltos:** Casos prácticos con soluciones explicadas.  
✅ **Diapositivas de teoría y práctica:** Material utilizado en clase.
